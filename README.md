# AI Interview Platform

An AI-powered interview platform with a **React + Vite** frontend and a **Spring Boot** backend.

## Project Structure

```
AI-Interview-Platform/
│
├── frontend/        # React + Vite (client)
├── backend/         # Spring Boot (REST API)
├── docs/            # Documentation
├── README.md
└── .gitignore
```

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
./mvnw spring-boot:run
```
